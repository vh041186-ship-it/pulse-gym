# 💪 Pulse Gym

Landing page desenvolvida para a academia **Pulse Gym**, com foco em conversão de clientes e presença digital profissional.

---

## 🚀 Tecnologias

HTML
CSS
JavaScript

## 📌 Funcionalidades

* Apresentação da academia
* Layout responsivo (mobile e desktop)
* Menu toggle com sidebar
* Seções organizadas (planos, benefícios, contato, etc.)
* Navegação simples e intuitiva

---

## 📲 Contato

A landing page pode incluir botões de contato direto (como WhatsApp ou formulário), facilitando a conversão de visitantes em clientes.

## 🎯 Objetivo do projeto

Este projeto foi desenvolvido para simular uma landing page real de academia, com foco em:

* Atrair novos alunos
* Apresentar serviços de forma clara
* Melhorar a presença digital do negócio
* Praticar desenvolvimento front-end

---

## 💡 Possíveis melhorias

* Integração com backend (cadastro de alunos)
* Sistema de login
* Integração com pagamentos
* Animações mais avançadas
* SEO e otimização de performance
